## Build and Setup

The project requires [Udacity's simulator](https://github.com/udacity/self-driving-car-sim/releases).

```
1. Make a build directory: `mkdir build && cd build`
2. Compile: `cmake .. && make`
3. Run it: `./pid`. 
4. Start the simulator. 
```
